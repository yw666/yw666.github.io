Contribute

How to contribute

EC is an open-source, community-driven project. There are many ways to get involved.

🐛
File Issues
Found a bug or inconsistency? Open a GitHub issue with reproduction steps. We triage daily and aim to respond within 48 hours.
🔀
Submit PRs
New attack implementations, defense integrations, documentation fixes — all welcome. Check our contributor guide before starting.
📊
Add Benchmarks
Run EC benchmarks on a new model or dataset and submit results to our leaderboard. Rigorous evaluation methodology required.
📝
Write Tutorials
Help new researchers get started. We accept community-authored tutorials in our docs under your name and affiliation.
🌐
Translate Docs
Help make EC accessible globally. We currently have partial translations in Chinese, French, and German — more are needed.
🏫
Teach EC
Using EC in your course? We provide lecture slides, homework sets, and grading rubrics for free. Register as an academic partner.
